Вход в админ панель
    Путь URL /admin
    Login: admin
    Password: admin
Настройки подключения к базе данных расположены в классе config/BbConfig
В файле config/config.php - базовый url сайта и путь к папке с ресурсами (js, css, картинки и т.д.)

Для применения миграций перейти по ссылке /admin/migrationsUp
Для отмены миграций перейти по ссылке /admin/migrationsDown
